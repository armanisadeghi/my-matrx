About This Project

Learn more about the My Matrx repository structure and how to use it.

Purpose

This repository is designed to be a clean, simple foundation for creating and deploying individual HTML pages with CSS and JavaScript on Vercel.

Structure

  • pages/ - Individual HTML pages
  • assets/ - Shared CSS, JS, and images
  • templates/ - Page templates for quick creation

Getting Started

Copy a template from the templates folder to the pages folder, customize it, and deploy!
